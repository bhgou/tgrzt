# Demo Stage Bot

Telegram-бот с Mini App для музыкантов: загрузка демо в WAV, автоматическая конвертация в MP3 для прослушивания, рейтинги по 10-балльной шкале, лайки, комментарии, поиск артистов, подписки и личный кабинет.

## Что внутри

- `Главная` с новыми и рейтинговыми треками
- `Личный кабинет` с регистрацией, профилем, лайкнутыми и своими треками
- поиск артистов и треков
- загрузка `.wav` с конвертацией в `.mp3`
- оценки `1-10`, лайки, комментарии, подписки на артистов
- Telegram WebApp auth через `initData`
- dev-режим без Telegram, если `ALLOW_DEV_AUTH=true`
- **Админ-панель** для управления пользователями и назначения администраторов

## Запуск

1. Скопируй `.env.example` в `.env` и заполни:
   - `BOT_TOKEN` от BotFather
   - `APP_BASE_URL` с публичным `https://` адресом Mini App
2. Установи `ffmpeg`, иначе загрузка WAV не сможет конвертироваться в MP3.
3. Запусти:

```bash
npm start
```

Проект использует только встроенные возможности Node.js, поэтому внешние npm-зависимости не нужны.

## Telegram

Бот работает через long polling. После `/start` он отправляет кнопку для открытия Mini App. Для реального запуска в Telegram `APP_BASE_URL` должен быть публичным HTTPS-адресом.

## Важно

- без `ffmpeg` сервер поднимется, но загрузка треков вернёт понятную ошибку
- локально без Telegram будет использован dev-пользователь
- данные и медиа складываются в `data/`

## Админ-панель

Админ-панель доступна по адресу `/admin.html` и позволяет:

- **Просматривать всех пользователей** системы с поиском по имени, username или Telegram ID
- **Управлять администраторами**: добавлять и удалять админов по Telegram ID
- **Просматривать список текущих администраторов**

### Доступ к админ-панели

**Через Telegram Mini App:**

1. Убедитесь, что ваш Telegram ID указан в `ADMIN_TELEGRAM_IDS` в файле `.env`
2. Откройте Mini App через Telegram
3. Перейдите в `Личный кабинет` - там появится кнопка "Открыть админ-панель"
4. Нажмите на неё для перехода в админ-панель

**Прямая ссылка (для разработки):**

Вы также можете открыть `http://localhost:3000/admin.html` или `https://ваш-ngrok-url/admin.html` в браузере, но для работы нужна Telegram аутентификация через `X-Telegram-Init-Data` заголовок.

### API endpoints

Админ-панель использует следующие API endpoints (доступны только администраторам):

- `GET /api/admin/check` - проверка прав администратора
- `GET /api/admin/users` - получение списка всех пользователей
- `GET /api/admin/users/admins` - получение списка администраторов
- `POST /api/admin/users/:id/admin` - назначение пользователя администратором
- `DELETE /api/admin/users/:id/admin` - удаление прав администратора
- `POST /api/admin/link-artist` - привязка артиста к Telegram аккаунту
- `POST /api/admin/transfer-artist` - перенос данных артиста между пользователями

### Привязка артиста к Telegram аккаунту

Если вы зарегистрировались с новым Telegram аккаунтом, но у вас уже есть артист с данными (треки, подписчики, оценки), вы можете привязать его к текущему аккаунту:

1. Откройте админ-панель
2. Перейдите на вкладку **"Привязка артиста"**
3. Введите **Telegram ID** аккаунта, к которому нужно привязать артиста
4. Введите **никнейм артиста**, чьи данные нужно перенести
5. Нажмите **"Привязать артиста"**

**Что происходит:**
- Все треки, подписчики, оценки, лайки и комментарии переносятся на новый аккаунт
- Старый пользователь удаляется из базы данных
- Новый аккаунт получает все данные артиста
